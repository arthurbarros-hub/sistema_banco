# Sistema Banco - Projetos

## Visao geral
Repositorio com tres scripts em Python. Cada arquivo e um projeto independente com menu interativo no terminal.

## Projetos
### 1) Sistema bancario
Arquivo: 1/Arthur&Fernando.py

Funcionalidades:
- Menu com saldo, deposito, saque e extrato
- Simulacao de investimento com taxa mensal
- Porquinho com ganho diario fixo

### 2) Calculadora de funcoes
Arquivo: 2/Isaque&pedro.py

Funcionalidades:
- Operacoes basicas (soma, subtracao, multiplicacao, divisao)
- Media, equacao de segundo grau, fatorial
- Tabuada e sequencia de Fibonacci

### 3) GeoQuiz
Arquivo: 3/Marcos&Nicolas.py

Funcionalidades:
- Quiz de geografia com perguntas e alternativas
- Pontuacao final ao terminar as perguntas

## Como executar
Requisitos: Python 3 instalado.

No terminal, execute um dos comandos:

```bash
python 1/Arthur&Fernando.py
```

```bash
python 2/Isaque&pedro.py
```

```bash
python 3/Marcos&Nicolas.py
```

## Observacoes
- Cada script roda de forma independente.
- Use o menu de cada programa para navegar pelas opcoes.
